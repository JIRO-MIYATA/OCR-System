# Smart Document OCR - 簡易マニュアル

本システムは、PDF、Word、Excel、画像（JPEG/PNG）などの様々なドキュメントから、AI（Gemini API）を活用して必要なデータを自動抽出し、Excelデータベースへ一元管理・蓄積するアプリケーションです。

---

## 🚀 1. セットアップと起動方法

### ① 動作環境の準備
本システムは Python で動作します。すでに仮想環境（`.venv`）が準備されています。

1. **コマンドプロンプトまたは PowerShell** でプロジェクトフォルダに移動します。
2. 以下のコマンドで仮想環境を起動します。
   * **PowerShell**:
     ```powershell
     .venv\Scripts\Activate.ps1
     ```
   * **コマンドプロンプト (cmd)**:
     ```cmd
     .venv\Scripts\activate.bat
     ```
3. 依存ライブラリがインストールされていない場合は、以下を実行します。
   ```bash
   pip install -r requirements.txt
   ```

### ② API キーの設定
Gemini API を使用するために API キーが必要です。以下のいずれかの方法で設定できます。
* **推奨方法 (.env ファイル)**:
  プロジェクトのルートディレクトリにある `.env` ファイルに以下のように記述します（`.env.example` をコピーして作成できます）。
  ```env
  GEMINI_API_KEY=あなたのGemini_APIキー
  ```
* **画面上での入力**:
  システム起動後、ブラウザ画面のサイドバーから一時的に API キーを入力することも可能です。

### ③ アプリケーションの起動
以下のコマンドを実行して Streamlit サーバーを起動します。
```bash
streamlit run app.py
```
起動すると、ブラウザが自動的に開き `http://localhost:8501` にアクセスします。

## 🖥️ 2. 画面操作・使用方法

システム画面は、左側の **「サイドバー（システム設定）」** と、右側の **「メインパネル（解析と一覧）」** で構成されています。

### ① サイドバー（システム設定）
1. **Gemini APIキー**: 
   環境変数（`.env`）からロードされている場合は「ロードしました」と緑色のバッジが表示されます。未設定の場合は、ここに直接 API キーを入力してください。
2. **抽出する項目**: 
   抽出したいデータ項目を**カンマ（`,`）区切り**で入力します（例: `日付, 取引先, 金額, 品目, 備考`）。
   * *※解析を実行する前に設定してください。*
3. **データベースをリセット**:
   蓄積された Excel データをすべて削除し、初期状態に戻すことができます。

### ② メインパネル - タブ1:「ファイルのアップロードと解析」
1. **ファイルのアップロード**:
   * 点線で囲まれた**「ドラッグ＆ドロップゾーン」**にファイルをドロップするか、中央の「Upload」ボタンをクリックしてファイルを選択します（複数ファイルを同時に選択して一括で処理できます）。
2. **AIで一括解析を実行**:
   * 青い「🚀 AIで一括解析を実行」ボタンをクリックすると、順次解析が始まります。
   * 各ファイルの解析ステータスと、抽出された JSON データが画面にリアルタイムで表示されます。

### ③ メインパネル - タブ2:「蓄積データの一覧 & データ管理」
1. **全蓄積データの確認とダウンロード**:
   * これまでに解析し、抽出に成功した全データがテーブル形式で一覧表示されます。
   * 「📥 すべてのデータをExcelでダウンロード」ボタンから全件を一括ダウンロードできます。
2. **データの期間指定操作（抽出・削除）**:
   * **期間の指定**: 「開始日」と「終了日」を指定して、ファイルの処理日時を基準に対象データを絞り込みます。
   * **期間指定抽出**: 指定した期間に該当するデータのみをテーブルでプレビュー確認でき、ExcelまたはCSV形式を選択して個別にダウンロード可能です。
   * **期間指定削除**: 削除対象となるデータを事前にプレビュー確認できます。誤削除を防ぐため、「削除への同意」チェックボックスをオンにすることで、対象データのみを安全に削除できます。

## 💡 3. トラブルシューティング

* **APIキーエラーが発生する場合**
  * 入力した API キーが正しいか、スペースなどが混入していないかご確認ください。
  * APIキーの発行は [Google AI Studio](https://aistudio.google.com/) から行えます。
* **解析が「失敗」になる場合**
  * ファイル形式が対応しているか確認してください（PDF, DOCX, XLSX, XLS, JPG, PNG）。
  * ドキュメントの文字が極端に潰れている、または画像が暗すぎる場合、AI が読み取れないことがあります。
* **Excelファイルが壊れている、または保存できない場合**
  * `extracted_data.xlsx` が他のアプリケーション（Excelなど）で開かれたままになっていると、プログラムから書き込みができずエラーになります。ファイルを開いている場合は閉じてから再実行してください。
